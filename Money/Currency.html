    def exponent : Int32 #

    Returns the relation between subunit and unit as a base 10 exponent.

    Money::Currency.find(:usd).exponent # => 2
Money::Currency.find(:btc).exponent # => 8

    NOTE MGA and MRU are exceptions and are rounded to 1.

    See Active_codes.
